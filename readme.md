# **安装python 3.10**


## 1. 配置环境 ##

### 1.1 安装Python ###

[下载安装python 3.10](https://www.python.org/ftp/python/3.10.11/python-3.10.11-amd64.exe)


### 1.2 配置pip内地源，下载更加快 ###

`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple`
 

### 1.3 安装pycharm ###


### 1.4 在pycharm 创建虚拟环境 ###

### 1.5 在虚拟环境里安装依赖 ###

`pip3.10 install -r ./requirements.txt`
